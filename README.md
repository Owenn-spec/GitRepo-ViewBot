**How to Use.**


**Prepare Proxies:**
Import you're proxies into the **proxies.txt** file and list your proxies in the format **ip:port**, **one per line**.


**Set URL:**
Replace **"your-link"** in the script with the URL you want to make requests to.


**Install Requirements:**
Before running the script, ensure you have the required dependencies installed. You can do this by running:
**pip install -r requirements.txt**


**Customize Parameters:**
Adjust num_requests to specify the desired number of requests.
Set concurrency_limit according to your proxy provider's limits.


**Run the Script:**
Execute the script with **python Request.py**.
